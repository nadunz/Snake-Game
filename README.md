# Snake-Game
Simple playable snake game

## Gameplay
your objective is to devour any red foods without running into your tail
  - use arrow keys to navigate
    
## Requirements
  - jdk 1.8

# Compile & Run
  - $ `javac Game.java`
  - $ `java Game`
